# 3BEK Labs HTML site prototype

Статический HTML-прототип сайта консалтингового агентства.

Сайт опубликован на [3bek.ru](https://3bek.ru) через GitHub Pages.

## Страницы

- `index.html` — главная страница
- `ai-audit.html` — посадочная AI / LLM аудит
- `llm-model-adaptation.html` — посадочная по LLM/ML и дообучению моделей
- `production-ai-iot.html` — посадочная по production AI, backend и IoT
- `gtm-marketing.html` — посадочная по GTM, CRM и маркетингу
- `automotive-ai.html` — вертикальная страница automotive / fleet / mobility
- `healthtech-ai.html` — вертикальная страница HealthTech
- `cases.html` — кейсы и доказательства

## Внесённые правки

- Убрано `Tech Lead` из карточки Фарруха.
- Исправлено `Назар`.
- Ключевые цифры сформулированы через `устройства`, без сужения только до одного типа транспорта.
- Для Камиллы добавлены рынки: СНГ, ЮВА и LATAM.
- В основу блоков положены реальные достижения из резюме: highload/IoT, ML/LLM/HealthTech, CRM/GTM/marketing.

## Публикация

Изменения в ветке `main` автоматически публикуются workflow-файлом
`.github/workflows/pages.yml`. Домен задаётся в `CNAME` и настройках GitHub Pages.
