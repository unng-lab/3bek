# 3BEK Labs — standalone HTML build v3

Эта версия специально подготовлена для предпросмотра отдельных HTML-файлов в ChatGPT и браузере через file://.

- CSS встроен внутрь каждой страницы.
- JavaScript встроен внутрь каждой страницы.
- Reference design image встроен в design-board.html как data URI.
- Для просмотра не нужен локальный сервер и не требуется папка assets.

Главная: index.html

## Публикация

Сайт опубликован на [3bek.ru](https://3bek.ru) через GitHub Pages.
Изменения в ветке `main` автоматически публикуются workflow-файлом
`.github/workflows/pages.yml`. Домен задаётся в `CNAME` и настройках Pages.
